# Customer Churn Prediction ML

A basic machine learning project to predict customer churn for online streaming services like Netflix, Amazon Prime Video, Jio TV, and others. This project demonstrates data preprocessing, model training, evaluation, and prediction using Python's scikit-learn library.

---

## Features

- Data Cleaning and Preprocessing
- Label Encoding for categorical variables
- Train/Test Split
- Logistic Regression Model
- Random Forest Classifier
- Model Evaluation (Accuracy and Classification Report)
- Sample Prediction for a new customer

---

## Technologies Used

- Python 3.14
- pandas
- scikit-learn
- Jupyter Notebook
